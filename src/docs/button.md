# Button 按钮
----
## 基础用法
使用```type```、```plain```和```round```属性来定义 Button 的样式。

    <l-button>默认按钮</l-button>

    <l-button type="info">信息按钮</l-button>
    <l-button type="warning">警告按钮</l-button>
    <l-button type="danger">危险按钮</l-button>
  </div>
  <div style="paddingTop:10px;">
